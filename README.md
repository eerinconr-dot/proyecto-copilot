# Blog Técnico - Arquitectura de Software

Autor: Eber Rincon

Descripción: Proyecto de práctica que contiene un Blog Técnico con 3 posts sobre Arquitectura de Software. Implementado con HTML y CSS (JavaScript opcional).

Estructura:

- `index.html` - Página principal con enlaces a los posts.
- `posts/post1.html` - ¿Qué es la Arquitectura de Software?
- `posts/post2.html` - Estilos Arquitectónicos más utilizados
- `posts/post3.html` - Patrones Arquitectónicos Modernos
- `css/styles.css` - Estilos del sitio

Cómo ver el sitio localmente:

1. Abrir `index.html` en el navegador (doble clic o arrastrar al navegador).

Despliegue en GitHub Pages:

1. Inicializar repositorio (si aún no lo has hecho):

```
git init
git add .
git commit -m "Inicial: Blog Técnico - Arquitectura de Software"
```

2. Crear un repositorio en GitHub con el nombre que prefieras (por ejemplo `proyecto-copilot`), luego añadir el remoto y subir:

```
git remote add origin https://github.com/<tu-usuario>/<tu-repo>.git
git branch -M main
git push -u origin main
```

3. En GitHub: ir a `Settings` > `Pages` y seleccionar la rama `main` y la carpeta `/ (root)` como fuente. Guardar. GitHub Pages publicará el sitio en `https://<tu-usuario>.github.io/<tu-repo>/`.

Notas y siguientes pasos:

- Puedes mejorar el estilo, añadir imágenes y usar JavaScript para navegación dinámica.
- Si quieres, puedo crear un archivo `index.md` o preparar la configuración para Jekyll/Gatsby/Vite si prefieres un generador estático.
